Cheshire Configuration Files

[ Up to Cheshire Overview ]

This document aims to aid in configuring the Cheshire2 search engine by hand, in conjunction with the technical documentation. A sample configuration file is available here. It assumes that the cheshire installation is as described in the setup file.

Before continuing, please be aware that there is an excellent graphical tool available in order to configure and maintain the configuration files. This is (currently) in the source code for cheshire, in the directory 'config-gui'. To start it, chmod 'sc.tcl' to 755 and run it.

The configuration for each Cheshire database is in the form of an SGML document. It has the basic structure as shown in the tree page. Have a look at this, as we'll be stepping through it bit by bit.
