Configuration Tree

The configuration for each Cheshire database is in the form of an SGML document. It has the following basic structure:

  • dbconfig
    • dbenv
    • filedef
      • filetag
      • filename
      • filecont
      • explain
        • titlestring
        • description
        • disclaimers
        • news
        • hours
        • besttime
        • lastupdate
        • updateinterval
          • value
          • units
        • coverage
        • proprietary
        • copyrighttext
        • producercontactinfo
          • contact_name
          • contact_address
          • contact_email
        • suppliercontactinfo
          • contact_name
          • contact_address
          • contact_email
      • filedtd
      • sgmlcat
      • assocfil
      • history
      • indexes
        • indexdef
          • indxname
          • indxtag
          • indxmap
            • use
            • relat
            • posit
            • struct
            • trunc
            • compet
          • stoplist
          • indxkey
            • tagspec
              • ftag
              • s
              • attr
              • value
      • clusters
        • clusterdef
          • clusname
          • cluskey
            • tagspec
          • stoplist
          • clusmap
            • from
              • tagspec
            • to
              • tagspec
            • summarize
              • maxnum
              • tagspec
      • components
        • componentdef
          • componentname
          • componentnorm
          • compstarttag
          • compendtag
          • componentindexes
            • indexdef
      • dispoptions
      • displays
        • displaydef
          • convert
            • clusmap
          • exclude
            • tagspec

The clusmap, indexdef and tagspec trees are only shown once on the tree.
