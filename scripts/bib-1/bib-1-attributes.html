Bib-1 USE attributes

The USE attribute determines which index on the server is searched. The following set of menus include the Z39.50 BIB-1 options for USE values.

Original Bib-1                              WAIS Additions                             Approved Extensions

        

Dublin Core USE attributes             GILS Attributes                         New Music Attributes

             

A complete description of the BIB-1 attributes and semantics can be found at the Z39.50 Maintenance Agency

The Dublin Core Semantics are defined in Dublin Core Metadata Element Set: Reference Description.

The GILS Use attributes, approved at the June 1998 ZIG meeting, correspond semantically to GILS Core Elements, described in the GILS Z39.50 Specification Annex E.
