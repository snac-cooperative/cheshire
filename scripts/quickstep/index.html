Search Field Information

  • Anywhere: This will search across the entire message, both headers and body.
  • Author: This will search for author, either by name or email address.
  • Date: This is for searching for recent messages.
  • Start Month: The month to start searching at.
  • End Month: The month to end searching after.
  • Subject: Terms here will be searched for in the subject line.
  • Message Body: This is for searching the actual text of the message.
  • Combine Fields With: If multiple fields are given, then they will be combined with this.
  • Format:
    • Summary will display just the subject, author and date, with a link to the full message
    • Full Message will display all of the message for each matched.
    • Threads will display in the same manner as Summary, but only the first message in each thread
  • Records to Display: This is the number of records that will be displayed per page
  • Start Record: This is the number of the first record to be displayed.
