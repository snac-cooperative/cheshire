Using EadWiki

Introduction

EAD Wiki is a system to allow online creation and editing of EAD records, using both full SGML editing and an ISAD(G) style template. Record metadata is recorded, as is information as to who is permitted to edit each record.

Viewing Records

Type in the URL of the location of the record that you wish to see. In the future there will be facilities to search for individual records and browse the existing records. For the moment, write down the location of each record that you create.

Editing a Record

  1. To edit a record, you need to call it up in HTML first as above.
  2. At the bottom of the page will be a button called 'Edit', click on this.
  3. A window will appear asking for a name and password and you need to type these in.
  4. A selection of commands will appear, ensure that 'edit record' is selected and click submit
  5. The template will appear, with the values filled in. Edit these entries and click Render to SGML at the bottom of the file.
  6. A second window will appear. The EAD record will be inside a box, which can be further edited if desired.
  7. When the record is updated, click on the Save button.

    Creating a New Record

    Currently Creating a new record directly from a location doesn't work fully. Instead take an existing record and copy it to a new location, then edit it from there.
    To do this:

    • Call up the record in HTML
    • Click the Edit button
    • Select 'Copy' from the list of options
    • A form will appear, select a new location and identifier. The location is where the record will be, the ID is to keep the record distinct from any other record. When you have filled these out, click Submit and the record will be copied.

    Error Messages

    There are several error messages that may appear. For example you cannot copy a record over an existing one. In these cases, you'll be prompted to fill out the form again.
